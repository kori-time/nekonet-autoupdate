# Read-only API reference

- `GET /health`
- `GET /api/v1/version`
- `GET /api/v1/status`
- `GET /api/v1/fleet`
- `GET /api/v1/servers`
- `GET /api/v1/servers/{id}`
- `GET /api/v1/history`
- `GET /api/v1/history/{run_id}`
- `GET /api/v1/events`
- `GET /api/v1/storage`
- `GET /api/v1/network`
- `GET /api/v1/certificates`
- `GET /metrics`
- `WS /ws`

There are no public run-start, cancellation, fleet-write, configuration-write,
or certificate-write endpoints.
